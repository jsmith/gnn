# GNN
Golang Neural Network (GNN) framework!

1. Network is given {  
  Layer,  
  Activation,  
  ...  
  Layer,  
  Activation  
}
2. Trainer is given {  
  GradientDescent,  
  BatchSize,  
  Cost,  
  UpdateFunc,  
  LearningRate  
}
3. Run the Trainer with # of iterations

*[anynet](https://github.com/unixpickle/anynet) heavily used as reference*
